-- PROGRAM BINUS PARKING --

- Gunakan "admin" (tanpa tanda kutip) untuk username dan password
- Input LISENCE PLATE tanpa spasi
- Menggunakan single linked list untuk menyimpan data kendaraan yang terparkir
- Menggunakan binary search tree untuk menyimpan data riwayat parkiran (berisi data kendaraan yang telah keluar dari parkiran)
- Kalkulasi biaya parkir meliputi:
	- mobil -> Rp5000 untuk 5 detik pertama pertama, Rp3000/5detik untuk detik berikutnya
	- motor -> Rp2000 untuk 5 detik pertama, Rp1000/5detik untuk detik berikutnya
